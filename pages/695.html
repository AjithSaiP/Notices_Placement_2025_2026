Job Description: About the Role :
We are developing an advanced software stack that enables the integration of mathematical modeling, circuit simulation, and embedded logic for edge computing applications in energy, control, and automation. As an intern, you will contribute to building the software tools and infrastructure that drive simulation pipelines, embedded control workflows, and hardware interfacing. The work involves circuit modeling, object-oriented programming, data-driven workflows, and software abstractions that support digital and analog system behavior. You’ll help prototype and validate software systems that interact with physical electronics and real-time computing platforms.

What You’ll Do :
● Work with electronic components such as transistors, Op-Amps, FETs, and
passive/active circuit elements
● Simulate and model circuit behavior using tools like LTspice, NGSpice, or equivalent
● Develop software tools to automate circuit generation, simulation workflows, and result parsing
● Implement firmware or system code on microcontrollers (e.g., STM32, ESP32) and edge devices (e.g., Jetson, Raspberry Pi)
● Prototype embedded logic or control blocks using FPGAs or FPAAs
● Use object-oriented programming (OOP) principles to build maintainable and reusable simulation and control frameworks
● Benchmark system performance across metrics like latency, stability, power
consumption, and accuracy

Required Skills :
● Strong understanding of circuit fundamentals, including behavior of transistors (BJT,
MOSFET) and operational amplifiers (op-amps)
● Experience using circuit simulation software (LTspice, NGSpice, or similar)
● Proficiency in Python (including libraries such as NumPy, SciPy, matplotlib, pandas)
● Working knowledge of C/C++ for embedded or systems programming
● Understanding of object-oriented programming (OOP) and modular software design

Nice to Have :
● Hands-on experience with HDL (Verilog/VHDL) or FPGA-based prototyping
● Previous work in hardware-software co-simulation or hardware-in-the-loop (HIL) setups
● Interest in analog-digital hybrid systems, reconfigurable computing, or unconventional computing models

What You’ll Gain :
● Deep understanding of physical computing and hardware-accelerated system design
● Hands-on experience with integrated analog, digital, and software-based computation
● Exposure to real-world applications in energy, optimization, and embedded intelligence
● Close mentorship and the opportunity to contribute meaningfully to a research-grade product
